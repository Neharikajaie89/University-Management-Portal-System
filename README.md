# University Management Portal System

## Overview

The University Management Portal System is a basic front-end project developed using **HTML only**. It demonstrates the structure and layout of a university portal, providing static web pages for common university management functions.

This project is intended for educational purposes and helps beginners understand HTML page structure, navigation, and form creation.

## Features

- Home Page
- About University
- Student Registration Form
- Student Login Page
- Faculty Information
- Course Details
- Admission Page
- Contact Us Page
- Navigation Menu
- Static HTML Forms

## Technologies Used

- HTML5

## How to Run

1. Download or clone the project.
2. Open the project folder.
3. Double-click `index.html` or open it in any modern web browser.
4. Navigate through the pages using the menu.

## Purpose

This project was created to practice:

- HTML document structure
- Hyperlinks and navigation
- HTML forms
- Tables
- Lists
- Images
- Semantic HTML elements

## Limitations

- No CSS styling
- No JavaScript functionality
- No database integration
- No backend functionality
- Static web pages only
